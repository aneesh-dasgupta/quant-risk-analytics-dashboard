# Cross Asset Portfolio Analytics Dashboard

## Overview

Institutional-style portfolio analytics engine built using Python that evaluates equities, ETFs, commodities, FX and cryptocurrencies through a unified quantitative risk framework.

## Features

• Automatic market data collection using Yahoo Finance API
• Portfolio return analytics
• Sharpe Ratio
• Sortino Ratio
• Calmar Ratio
• Maximum Drawdown
• Rolling Sharpe
• Correlation Heatmap
• Risk Return Scatter Plot
• Interactive Plotly Dashboard

## Technologies

Python
Pandas
NumPy
Plotly
Yahoo Finance API
Matplotlib
